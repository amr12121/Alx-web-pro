# Eltohamy Portfolio Website

This is a responsive portfolio website created to showcase Amr Eltohamy's skills as a Front-End Developer. The website includes a homepage with social media links, contact details, and sections for skills.

## Features
- **Responsive Design**: The layout adapts to different screen sizes, ensuring the website looks great on mobile, tablet, and desktop devices.
- **Smooth Scrolling**: Users can smoothly scroll between sections of the website.
- **Contact Section**: Provides phone and email details.
- **Social Media Integration**: Links to LinkedIn, Twitter, YouTube, and Instagram.
- **Animated Hover Effects**: Cards and buttons have hover animations that enhance the user experience.

## Technologies Used
- **HTML5**
- **CSS3**
- **FontAwesome** for icons
- **Google Fonts (Poppins)** for typography

## Installation
1. Clone the repository to your local machine using:
    ```bash
    git clone https://github.com/yourusername/portfolio-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio-website
    ```
3. Open `index.html` in your preferred browser.

## Sections
1. **Header**: 
   - Contains the logo and navigation links.
   - Stays fixed at the top as you scroll.

2. **Main Section**: 
   - Includes an introduction with a welcome message and social media links.
   - Background image with a fixed scroll effect.

3. **Contact Section**: 
   - Users can find Amr's phone number and email address.
   - Includes contact icons with hover effects.

4. **Footer**: 
   - Copyright information.
   - Social media links repeated for easy access.

## How to Customize
- Update the `index.html` file with your personal information.
- Modify the `style.css` file to change colors, fonts, or layout.
- You can change the social media links to your own profiles in the HTML.
